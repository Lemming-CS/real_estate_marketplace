# Demo Video Checklist

Recommended 6-10 minute demo order:

1. Introduce the product as a real-estate marketplace for renting and buying apartments/houses.
2. Show repository structure quickly.
3. Start backend/admin/mobile locally.
4. Login as renter and browse the property feed.
5. Show search, sort, and KGS-based price filtering.
6. Open a property detail page:
   - photo gallery
   - map/location
   - seller card
   - favorites/views counters
7. Start a conversation with the seller and send a message with attachment.
8. Show notifications and unread badge updates.
9. Login as seller:
   - create or edit a property
   - show map pin placement
   - show optional video/media support
   - mark a listing as sold
10. Show promotion purchase history / active promotion state.
11. Submit a listing report as a renter.
12. Switch to admin:
   - review report queue
   - open linked listing/user context
   - show suspension note visibility
   - show audit log entry
13. Close with known limitations and future work.

## Must Mention Verbally
- sellers publish directly after validation
- admin moderation is report-driven, not universal pre-approval
- map coordinates are supported
- property photos and optional video are supported
- promotion activation depends on successful payment
- suspension notes are stored durably and visible in admin detail
