# Screenshots Guide

Capture these screenshots before submission so the product reads clearly as a real-estate marketplace:

## Mobile
- home feed with mixed rent/sale property cards
- listing detail with photo gallery, counters, and map section
- listing create/edit form with property fields and map pin
- my listings showing `sold` and other owner-facing statuses
- inbox and conversation detail
- promote listing screen
- profile/settings with language switch

## Admin
- dashboard summary
- reports queue
- user detail showing suspension note history
- listing moderation view
- promotion packages and payments view
- scoped conversation review with attachment preview

## General Guidance
- use seeded real-estate data, not empty states only
- capture at least one Russian-language screen
- include one promoted listing
- include one reported listing / moderation screen
- include one map-focused screen
